## 📌 프로젝트 목표

- **어떤 문제를 해결하고 싶은가요? (Why)**
  - 개인의 성과를 시각적으로 축적·표현함으로써, 사용자가 스스로의 성취를 ‘눈으로 보는’ 경험을 제공, 할 일에 대한 동기를 제공하고 싶다.

- **그래서 무엇을 만들 건가요? (What)**
  - 사용자가 투두리스트를 작성하고, 각 항목을 완료할 때마다 해당 카테고리의 행성이 생성된다.
  - 완료된 행성은 ‘나’라는 태양 주위를 공전하며, 같은 카테고리의 할 일을 여러 개 완수할수록 행성이 합성되어 커진다.
  - 하루를 마감할 때, 사용자는 완성된 나만의 태양계를 GIF 형식으로 저장할 수 있다.

## 核心 기능 3가지

1. **기능 1:** 체크박스 형식의 투두리스트
2. **기능 2:** 완료된 업무를 을 행성으로 시각화하여 '나만의 태양계' 생성
3. **기능 3:** 하루의 성취를 GIF 헝식으로 개인화, 저장
